Create a web-based interactive visualization tool for learning about linked lists, stacks, and queues.
Key Features:

Interactive diagrams for each data structure.
Step-by-step visually appealing animations of common operations. Each action for respective data structure should follow proper format such as for stacks, push and pop would be the actions etc.
User input to modify and interact with the structures.
Quiz section to test understanding. Upon submission of quiz answers, the system should show them how many they got correct out of all the questions and visually show them which questions they got wrong vs correct. The quiz section should allow the users to retake the quiz as many times as they want.

Technical Requirements:

Single HTML file with HTML, CSS, JavaScript.
Clear, appealing, intuitive interface with smooth animations.
Accurate representations of data structure behaviors.
Error-free functionality and informative tooltips.
